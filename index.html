About

The Customised Pill Dispenser Tool addresses medication non-adherence in elderly and multi-medicated patients by automating dose release with accurate scheduling and verification. Built around an Arduino Uno, DS3231 RTC, a servo-driven 6-compartment rotating tray, and simple user feedback (LCD, LEDs, buzzer), the device is low-cost, modular, and easy to build.

Problem

Missed or incorrect doses lead to complications. Caregivers need a low-cost, reliable tool to reduce errors.

Solution

Automated timed dispensing with verification reduces human error and supports caregivers.

Benefits
  • Improves adherence
  • Simple UI — LCD, LEDs, buzzer
  • Expandable (IoT, remote monitoring)
